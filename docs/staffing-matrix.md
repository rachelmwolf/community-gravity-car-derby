# Staffing Matrix

## Purpose

This document turns the volunteer model into phase-based staffing coverage for the current event plan. It is designed for a first-year, volunteer-only community derby where role fit matters as much as raw headcount.

## Planning Assumptions

- current schedule assumes about 60 cars
- day-before setup runs from 3:00 PM to 7:00 PM on Friday, May 1, 2026
- event-day operations run from 8:00 AM to 3:00 PM on Saturday, May 2, 2026
- the event is trying to deliver polished execution without overspending volunteer capacity

## Core Principles

- critical decision roles should be assigned before general volunteer recruitment
- volunteers should choose roles based on fit, not just open slots
- every critical role should have identified backup coverage
- race-control roles should not be combined with hospitality roles unless attendance stays small

## Recommended Staffing Target For This Event

For the current 60-car planning assumption, this event should target:

- 3 critical leads assigned in advance:
  - Race Director
  - Chief Inspector
  - Safety Lead
- 5 additional core operating volunteers assigned in advance:
  - Event Host / MC
  - Track Operations
  - Check-In Lead
  - Staging Lead
  - Scoring / Results
- 5 to 7 flexible volunteers recruited by preference:
  - inspection assistant
  - check-in support
  - pit / repair support
  - finish / retrieval
  - floater / runner
  - setup / teardown support
  - hospitality / recruiting if staffing allows

This implies a realistic target of 13 to 15 volunteers for a smooth standard event, with 10 to 11 as an absolute minimum only if several people cover multiple phases.

## Role Fit Guidance

### Good Roles For Committed Core People

- Race Director
- Chief Inspector
- Safety Lead
- Track Operations
- Scoring / Results

These roles carry continuity, judgment, or technical load and should not be left to late volunteer assignment.

### Good Roles For Preference-Based Volunteers

- greeter / welcome
- check-in support
- inspection assistant
- staging runner
- finish / retrieval
- floater
- setup / teardown
- hospitality / recruiting

These are easier to describe clearly and easier for volunteers to choose based on comfort and energy level.

## Volunteer Signup Categories

To make staffing easier, volunteers should be asked to self-identify into categories before final assignment:

- guest-facing and welcoming
- organized and paperwork-focused
- handy / mechanical help
- tech / computer comfortable
- physically active / runner role
- safety / first-aid background
- flexible fill-in / floater

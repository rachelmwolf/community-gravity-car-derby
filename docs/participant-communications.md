# Participant Communications

## Purpose

Use this as the source for participant-facing email and message templates.

## Messaging Rule

Do not assume a participant has seen earlier emails.

Every important message should repeat the core facts:

- what the event is
- what the ticket includes
- what the participant needs to do next
- where to find build resources
- what happens at inspection

## Core Details To Repeat

- event: `Community Gravity Car Derby`
- date: `May 2, 2026`
- location: `The Church of Jesus Christ of Latter-day Saints, 150 Pine Ave, Colorado Springs, CO 80906`
- ticket: `$35` for one car kit and one race entry
- build model: build at home
- policy: all cars may race; only inspection-compliant cars are eligible for speed trophies; all cars remain eligible for design recognition

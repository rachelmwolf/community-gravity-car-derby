# Volunteer Quick Reference

## Critical Leads

These roles should be assigned before general volunteer recruitment.

| Role | Main Job | Best Fit | Shift |
| --- | --- | --- | --- |
| Race Director | make final decisions and keep the event on time | calm decision-maker | day before through teardown |
| Chief Inspector | enforce build rules and resolve pass/fail questions | detail-oriented, fair, steady | setup of inspection through reinspection window |
| Safety Lead | manage hazards, first aid, and escalation | safety-minded, calm | setup through teardown |
| Track Operations | run heats and keep racing moving | technical, steady, focused | race block |
| MC / Event Host | explain what is happening and keep energy up | clear speaker, welcoming | opening through awards |
| Scoring / Results | manage race results and support awards | accurate, computer-comfortable | race block through awards |

## Flexible Volunteer Roles

These are good roles for volunteers offering 1 to 3 hours.

| Role | Main Job | Good For |
| --- | --- | --- |
| Check-in support | greet racers, verify names, direct families | organized, guest-facing people |
| Inspection assistant | help measure and verify rule compliance | patient, detail-oriented people |
| Pit / repair support | help with minor compliant fixes | handy volunteers |
| Staging runner | move accepted cars into order for racing | organized, active volunteers |
| Finish / retrieval | return cars and reset flow | active volunteers |
| Floater | cover gaps, breaks, and short tasks | flexible helpers |
| Setup / teardown | move tables, signs, and equipment | volunteers available before or after the event |
| Hospitality / recruiting | support snacks, welcome, and recruiting table | friendly guest-facing volunteers |

## Shift Buckets

Use these when recruiting:

- `Day Before Setup`: Friday 3:00 PM to 7:00 PM
- `Event Morning`: Saturday 8:00 AM to opening ceremony
- `Race Block`: opening ceremony through finals
- `Awards / Teardown`: awards through closeout

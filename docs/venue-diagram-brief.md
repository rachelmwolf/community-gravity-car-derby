# Venue Diagram Brief

## Purpose

Use this brief to build the diagram volunteers will use for setup.

## Audience

- setup volunteers
- race-day leads
- first-time helpers with limited time

## The Diagram Must Be Easy To Read

- plain labels
- clear arrows
- no clutter
- one-page usable print version

## Required Labels

- entrance
- check-in
- inspection
- pit / repair
- impound / staging
- track start
- track finish
- scoring table
- seating / viewing
- awards area
- first aid
- emergency access

## Required Flow Arrows

- entrance to check-in
- check-in to inspection
- inspection to pit / repair if needed
- inspection to impound / staging if passed
- staging to track
- track to finish / retrieval
- family viewing path

## Nice To Have

- table counts
- signage markers
- power outlet notes
- storage area

## Volunteer Expectation

Any volunteer should be able to look at the diagram and answer:

- where do I set up my station
- where do families go next
- what area must stay clear

# Benchmark Findings

## Summary

Research across pinewood-derby-style events, formal soap box derby operations, and volunteer-management guidance points to the same conclusion: high-end execution comes from disciplined flow and clear role design.

## What Successful Derby-Style Events Do

- publish rules early
- preassign volunteer roles
- run a disciplined check-in and inspection process
- use a pit area for corrections
- move accepted cars into controlled staging
- keep race results objective and organized
- avoid dead air with a strong MC and clear family communication
- prepare awards before the ceremony

## What Successful Gravity Race Events Add

- named command group for decisions
- stronger inspection and safety controls
- controlled pit, pre-stage, course, and finish zones
- clear response path for incidents
- day-before setup and safety walkthrough
- active course and crowd monitoring

## Practical Recommendation For This Event

Use a hybrid model:

- family-friendly participant experience from strong pinewood derby practice
- operational discipline from gravity/soap box race practice

This is the right fit for a first-year community event that wants polished execution without overbuilding the production.

## Design Direction For This Repository

The repository should preserve:

- planning assumptions
- role definitions
- race-day run-of-show
- inspection and safety artifacts
- post-event lessons learned

That is the minimum documentation set needed to make the event repeatable.

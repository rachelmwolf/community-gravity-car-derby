# Gravity Car Derby Inspection Checklist

Use this at the inspection table. Keep one copy for the inspector and one visible copy for helpers.

## Station Ready

- [ ] Official scale ready
- [ ] Width gauge ready
- [ ] Ground-clearance gauge ready
- [ ] Starting-pin fit check ready
- [ ] Height limit reference ready
- [ ] Printed build rules available
- [ ] Pit / repair area ready
- [ ] Marking method ready for `Speed Eligible` or `Not Speed Eligible`

## Check Every Car

- [ ] Car does not extend beyond the starting pin
- [ ] Width is within limit
- [ ] Height is `4 inches` or less
- [ ] Ground clearance meets minimum
- [ ] Wheel track meets minimum
- [ ] Final assembled weight is at or under limit
- [ ] All four wheels touch
- [ ] Wheels and axles are standard kit parts or committee-approved same-type replacements
- [ ] No prohibited wheel reshaping
- [ ] Dry lubricant only
- [ ] No suspension
- [ ] No propulsion or launch mechanism
- [ ] No loose decorations or unsafe parts
- [ ] Weights are solid and secured

## Outcome

- [ ] `Speed Eligible`: car passes inspection and moves to impound / staging
- [ ] `Fix and Recheck`: send to pit / repair, then reinspect
- [ ] `Not Speed Eligible`: car may still race, but is ineligible for speed trophies

## Important Rule For Non-Compliant Cars

- non-compliant cars may still race
- non-compliant cars are not eligible for speed trophies
- non-compliant cars remain eligible for design recognition

## Escalate Immediately

- [ ] Rules dispute
- [ ] Unclear wheel or axle modification
- [ ] Car is close to the `4 inch` height limit
- [ ] Loose or unsafe build
- [ ] Any case not clearly covered by the printed rules

# Printables

## Purpose

This folder is for volunteer-facing documents that should be easy to print, export, or hand off outside GitHub.

## Recommended Structure

- keep a readable Markdown source in this folder whenever possible
- add exported PDFs here when the layout is finalized
- add Word documents only when a committee member truly needs that format
- use `live-links.md` to preserve links to any Google Docs or Google Sheets versions

## Current Items

- `inspection-checklist-printable.md`: one-page printable inspection checklist source
- `live-links.md`: place for Google Doc, Google Sheet, PDF, or shared-drive links

## File Naming

Use names like:

- `inspection-checklist-printable.pdf`
- `volunteer-check-in-sheet.docx`
- `day-of-run-sheet.pdf`

## Versioning Rule

If a printable file is updated outside GitHub:

- update the source in the repo if possible
- replace the exported PDF or Word file in this folder
- keep any live Google link current in `live-links.md`

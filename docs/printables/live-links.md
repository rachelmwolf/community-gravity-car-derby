# Live Links

## Purpose

Use this file to store links to live Google Docs, Google Sheets, shared-drive files, or exported PDFs that volunteers may need.

## Current Guidance

- add the live Google Doc link when a printable is actively edited there
- add the shared PDF link when a final printable is published outside GitHub
- keep the GitHub Markdown source whenever possible so future organizers can rebuild the file

## Suggested Entries

### Inspection Checklist

- GitHub source: `docs/printables/inspection-checklist-printable.md`
- Google Doc: add link when created
- PDF: add link when exported

### Day-Of Run Sheet

- GitHub source: `docs/day-of-run-sheet.md`
- Google Doc: add link when created
- PDF: add link when exported

### Volunteer Check-In Sheet

- Google Sheet or Doc: add link when created
- PDF or Word: add link when exported

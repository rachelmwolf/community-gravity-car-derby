# Venue Setup Plan

## Purpose

This is the working setup guide for volunteers preparing the venue.

## Setup Zones To Define

- arrival and welcome
- check-in
- inspection
- pit / repair
- impound / staging
- track and finish area
- scoring / software table
- seating / family viewing
- awards area
- recruiting / hospitality if used
- first-aid location

## Minimum Venue Questions To Answer

- where do families enter
- where does check-in happen
- where do cars go immediately after check-in
- where can on-site fixes happen
- where are passed cars held before racing
- where do families watch racing
- where is the software / scoring station
- where are awards presented
- where is first aid
- what path must stay clear for emergency access

## Volunteer Setup Sequence

1. Mark the track location and finish area.
2. Mark the check-in table location.
3. Mark the inspection station.
4. Mark the pit / repair area next to inspection.
5. Mark impound / staging.
6. Mark scoring / software table.
7. Mark seating / family viewing.
8. Mark awards area.
9. Mark first-aid location.
10. Confirm emergency paths stay clear.

## What The Final Diagram Should Show

- all named setup zones
- volunteer-only areas
- family-facing areas
- track direction
- flow from arrival to awards
- emergency access path
- where signage should go

## Setup Deliverables Still Needed

- simple venue diagram
- setup checklist by zone
- teardown checklist by zone
- signage list tied to locations

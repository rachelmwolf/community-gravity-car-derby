# Venue Layout Review Notes

## Purpose

This document captures the current venue-layout thinking as shown in the working diagram and records the key review notes so the layout does not get lost.

## Current Layout State

The working layout currently assumes:

- check-in near the east-side gym entry
- inspection on the east wall
- pit / repair near inspection
- track operations in front of the stage
- impound / staging near the track
- audience seating centered in the room
- a closer racer viewing area between the track and the main seating block
- standing / circulation walkways around the seating blocks
- recruiting and concessions on the west side
- first aid near the entry / check-in side

## What Already Looks Strong

- technical operations are separated from the main audience block
- audience seating is spread out instead of crowding the track
- recruiting and concessions do not interfere with check-in
- the racer viewing area is distinct from general seating
- circulation lanes are intentionally preserved

## Recommended Improvements

### 1. Tighten Inspection And Pit

- place pit / repair directly adjacent to inspection
- make the failed-inspection path obvious and short

### 2. Mark Controlled Areas Clearly

- label impound / staging as volunteer-only
- label track operations as volunteer-only

### 3. Clarify Entry Functions

- decide whether welcome and check-in are one table cluster or two
- if separate, label them separately

### 4. Clarify First Aid

- separate it visually from check-in so volunteers do not confuse it with registration
- consider labeling it `First Aid / Safety`

### 5. Add Seating Setup Notes

- note the target seating layout directly on the diagram
- current recommendation: two seating blocks, center aisle, about 238 chairs at 7 rows or 272 at 8 rows

### 6. Add Minimal Flow Arrows

- entry to check-in
- check-in to inspection
- inspection to pit if needed
- inspection to impound / staging if passed

## Suggested Seating Note

`Audience seating target: 2 blocks, 17 chairs wide per block, 7 rows standard or 8 rows max, with center aisle and side aisles preserved.`

## Suggested Next Version

The next diagram revision should:

- push pit / repair flush to inspection
- label controlled areas
- add seating note
- add a few simple flow arrows

At that point, the layout should be strong enough for volunteer setup use even without exact dimensions.

# Venue Setup Checklist

## Entrance And Signage

- signs from main south entrance directing people to the gym
- signs at north gym entry doors
- welcome / check-in tables set at gym entry
- line-control space kept clear

## Check-In

- check-in tables ready
- participant list ready
- pens, tape, forms, and car numbering ready

## Inspection And Pit

- inspection tables placed along east wall
- pit / repair tables beside inspection
- official scale ready
- inspection checklist printed
- rule sheet printed

## Track And Race Operations

- track placed in front of stage with volunteer access preserved
- scoring / software table placed with power access
- TVs / projector connected and tested
- impound / staging area marked
- retrieval path clear

## Audience And Awards

- family seating spread through center / west side
- close-view area reserved for current racers
- awards area placed in front of stage
- trophy table location set

## Support Areas

- concessions placed outside the main race-operations path
- recruiting table placed in visible but non-congested space
- first-aid location marked

## Safety

- emergency paths clear
- stage access for volunteers preserved
- no audience chairs blocking race operations
- no cords or tables crossing main traffic paths

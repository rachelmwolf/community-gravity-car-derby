# Quick Start

## Use These Files First

If you are helping with this event and do not have much time, start here:

1. `docs/day-of-run-sheet.md`
2. `docs/volunteer-quick-reference.md`
3. `docs/inspection-station-checklist.md`
4. `docs/staffing-matrix.md`

## What This Event Needs Most

- clear lead assignments
- enough volunteers to cover check-in, inspection, staging, scoring, and teardown
- inspection done consistently
- race-day flow that stays on time

## Minimum Organizer Decisions Still Needed

- final maximum height that clears the timing gate
- final wheelbase wording
- final policy on replacement parts
- what happens to cars that fail inspection but could still be shown for design

## Minimum Volunteer Target

For the current 60-car plan, target:

- 3 critical leads
- 5 core operations volunteers
- 5 to 7 flexible volunteers

## If You Only Have 30 Minutes

- confirm who owns Race Director, Chief Inspector, Safety Lead, Track Operations, and MC
- review the day-of run sheet
- review the inspection station checklist
- review open `P0` items in `docs/task-backlog.md`

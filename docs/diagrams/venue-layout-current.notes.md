# Venue Layout Current Notes

## What This File Is

`venue-layout-current.drawio.xml` is the saved editable draw.io file for the current working venue layout.

It reflects the venue assumptions known at the time it was saved, including:

- gym floor working assumption of about `94 ft x 60 ft`
- stage offset from the gym floor on the north side
- family entry through the north-side gym doors
- check-in near the entry
- inspection on the east wall
- pit / repair near inspection
- track operations in front of the stage
- audience seating centered in the room
- a closer racer viewing area

## What This Version Is Good For

- preserving the current zoning concept
- discussing setup with volunteers
- revising placements as room knowledge improves

## What This Version Does Not Guarantee

- exact dimensions
- final furniture counts
- final power locations
- final safety placement

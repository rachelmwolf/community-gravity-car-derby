# How To Use The Draw.io Venue Layout

## Purpose

This guide is for future volunteers or organizers who need to open, review, or update the saved venue layout.

## File To Use

- `docs/diagrams/venue-layout-current.drawio.xml`

## How To Open It

### Option 1: In draw.io

1. Go to [draw.io / diagrams.net](https://app.diagrams.net/).
2. Choose where to open the file from.
3. Open `venue-layout-current.drawio.xml`.

### Option 2: Import Into Another Tool

If you use another diagram tool that supports draw.io XML, import the file there and verify the layout before editing.

## What To Look At First

- where families enter
- where check-in happens
- where inspection and pit are placed
- where the track sits relative to the stage
- where audience seating goes
- what areas must stay clear

## How To Use It For Setup

- print or display the layout for setup volunteers
- walk the room and compare the diagram to the real space
- mark anything that does not match reality
- update the file after the walk-through so the saved version improves over time

## What To Change Carefully

- track position
- check-in location
- inspection and pit location
- seating block size
- awards area
- emergency access lanes

These affect multiple volunteer stations and should be reviewed as a group, not changed casually.

## Minimum Update Rule

If someone changes the layout, they should also update:

- `venue-layout-current.notes.md`
- any setup checklist that depends on the layout

## Plain-Language Goal

A first-time volunteer should be able to open the file and quickly understand:

- where each station belongs
- how families move through the room
- what space must remain open

# Community Gravity Car Derby

This repository is the operating backbone for the Pack 110 Community Gravity Car Derby and any future version of the event. It is intended to preserve planning decisions, volunteer structure, race-day operations, and reusable templates so a future organizer can run the event without relying on tribal knowledge.

## Current Event

- Event name: Pack 110 Community Gravity Car Derby
- Dates: May 1-2, 2026
- Location: The Church of Jesus Christ of Latter-day Saints, 150 Pine Ave, Colorado Springs, CO 80906
- Coordinator: Rachel Wolf

## Purpose

This is a first-year community event that mirrors the family-friendly structure of a pinewood derby while operating under a gravity car derby format. The year-one goal is disciplined execution, fair competition, and a repeatable model, not maximum scale.

## Operating Principles

- Build around mission-critical functions, not vague volunteer titles.
- Use a hard inspection and check-in gate before any race activity.
- Separate participant-facing hospitality from race-control functions.
- Match volunteers to role fit and preference whenever possible.
- Document decisions so future organizers can reuse and improve the model.
